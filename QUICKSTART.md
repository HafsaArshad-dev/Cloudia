# 🚀 CLOUDIA - Quick Start Guide

## ⚡ Get Started in 3 Steps

### Step 1: Open the App
Simply open `index.html` in your web browser. No build process required!

### Step 2: API Key (Already Configured)
Your API key is already set in `js/config.js`. The app is ready to use!

### Step 3: Start Using
- **Search**: Type a city name and press Enter
- **GPS**: Click the 📍 button to use your location
- **Toggle**: Use the buttons in the header for theme, sound, and contrast

## 🎨 Features Ready to Use

✅ Real-time weather data  
✅ 5-day forecast  
✅ Dark/Light mode  
✅ Temperature unit toggle (°C/°F)  
✅ GPS location detection  
✅ Favorite cities  
✅ Animated backgrounds  
✅ Responsive design  
✅ Accessibility features  

## 🔊 Optional: Add Sound Files

To enable ambient sounds, add these files to the `sounds/` directory:
- `rain.mp3`
- `thunder.mp3`
- `wind.mp3`

The app works perfectly without them!

## 📱 Test It Out

1. Open `index.html` in your browser
2. Allow location access when prompted (or search for a city)
3. Enjoy your beautiful weather app!

## 🐛 Troubleshooting

**No weather data?**
- Check your internet connection
- Verify the API key in `js/config.js`

**Location not working?**
- Make sure you've allowed location permissions
- Try searching for a city manually instead

**Sounds not playing?**
- Add sound files to the `sounds/` folder
- Check browser autoplay settings
