# Sound Files

Place your ambient sound files here:

- `rain.mp3` - Rain sound effect (looped)
- `thunder.mp3` - Thunder sound effect (looped)
- `wind.mp3` - Wind sound effect (looped)

## Where to Get Sounds

You can find free ambient weather sounds from:
- [Freesound.org](https://freesound.org)
- [Zapsplat](https://www.zapsplat.com)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## Note

The app will work without sound files. The sound toggle will simply not play any audio if files are missing.
