# Icons

Place your app icon files here:

- `favicon.ico` - Main favicon for the browser tab

## Creating a Favicon

You can create a favicon using:
- [Favicon.io](https://favicon.io) - Generate from text or image
- [RealFaviconGenerator](https://realfavicongenerator.net) - Comprehensive favicon generator

## Note

The app will work without a favicon. The browser will show a default icon instead.
