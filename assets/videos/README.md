# Weather Background Videos

Place your weather video files here with the following exact names:

## Required Video Files

- `sunny.mp4` - Clear sunny day
- `sunrise.mp4` - Morning sunrise
- `sunset.mp4` - Evening sunset
- `partlycloudy.mp4` - Partly cloudy day
- `partlycloudynight.mp4` - Partly cloudy night
- `overcastclouds.mp4` - Overcast/cloudy day
- `rainynight.mp4` - Rainy night
- `snowday.mp4` - Snowy day
- `snowynight.mp4` - Snowy night
- `fog.mp4` - Foggy conditions
- `nightsky.mp4` - Clear night sky
- `thunderstorm.mp4` - Thunderstorm

## Video Specifications (Recommended)

- **Format**: MP4 (H.264 codec for best compatibility)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Aspect Ratio**: 16:9
- **Duration**: 10-30 seconds (will loop)
- **File Size**: Optimize for web (aim for 2-5 MB per video)

## Notes

- Videos will automatically loop
- Videos are muted by default
- Videos fade smoothly between transitions (1-2 seconds)
- An overlay ensures text remains readable
